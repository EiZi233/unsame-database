## 2. 品牌故事与理念 (Brand Story & Philosophy)

### 为什么叫 "unSAME"？
"unSAME" 是一个自创词，源于过剩的"整活精神"和成为最奇怪游戏创作者的志向。核心宗旨是创造与市面上现有游戏"不一样"（not the same）的游戏，这个初衷至今未变。

灵感来源于怀念千禧年（2000s）小游戏大爆发的时代，每天都有新奇的、前所未见的游戏诞生。行动纲领是："想要玩到不一样的游戏，就要先做出不一样的游戏。" 通过不断尝试自己没见过的玩法和机制，来接近这个目标。

### 我们做什么样的游戏？
关键词：**怪诞 (Weird), 真诚 (Heartfelt), 前所未有 (Something you haven't played before)**。

创作方法：勇于实验，尤其是在不寻常的机制 (unusual mechanics)、非传统的操控 (unconventional controls) 和情感化叙事 (emotional storytelling) 方面。

## 3. 团队背景与成就 (Team Background & Achievements)

- **团队构成**: 3名核心成员 + 各领域的合作者 (contractors)。
- **核心优势**: 团队虽年轻（应届毕业生身份），但拥有极为丰富的实战经验（12年开发经验）。高产且专注：6年内创造了超过50款游戏原型/作品。
- **声誉与认可**: 从学生时代起就在独立游戏圈小有名气，几乎囊括了大学生能获得的所有独立游戏奖项。

## 4. 内容创作指导 (Content Creation Guidelines)

在为unSAME Studio创作任何内容时，必须遵守以下原则：

| 维度 | 指导原则 |
| :--- | :--- |
| **语言风格** | 幽默、充满活力、带点"怪咖"气质。对自己的"怪"感到自豪。 |
| **用词偏好** | 可使用"整活"、"怪游戏"、"不一样"等关键词。避免陈词滥调和过于商业化的营销术语。 |
| **视觉方向** | 强调"实验性"和"非传统"。突出游戏机制的独特性，而非仅仅是画面精美。 |
| **核心信息点** | 强调"前所未见"的体验；突出团队丰富的经验和获奖背景；传递"Fun"的核心感受。 |
