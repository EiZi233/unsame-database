# 游戏档案: PLUG IT IN

## 1. 核心身份与概念 (Core Identity & Concept)

- **游戏名称**: PLUG IT IN
- **itch.io页面**: [unsame.itch.io/plug-it-in](https://unsame.itch.io/plug-it-in)
- **一句话概括**: 一款关于将各种插头正确插入对应插座的，令人放松且手感"多汁"(juicy)的益智游戏。
- **核心体验**: 放松 (Relaxing), 治愈 (Feel Good), 休闲 (Casual), 有满足感 (Satisfying)。
- **核心玩法**: 像现实生活中一样，通过旋转和拖拽，策略性地规划空间，将所有插头都插到插座上。大的插头会阻挡下方的空间，玩家需要规划最优的插入顺序。
- **游戏目标**: 匹配正确的形状，插上所有插头，让一切都完美归位——Make everyone happy :)

## 2. 核心设计与玩法 (In-Game Design & Gameplay)

### A. 游戏玩法与机制 (Gameplay & Mechanics)

游戏的核心是**空间规划谜题**。玩家通过两个极简操作与游戏互动：

| 操作 | 功能 |
| :--- | :--- |
| **点击** | 旋转插头 (Click to Rotate Plug) |
| **拖拽** | 移动插头 (Drag to Move Plug) |

### B. 游戏特色与内容 (Features & Content)

| 特色 | 说明 |
| :--- | :--- |
| **"多汁"的反馈 (Juicy Feedback)** | 视觉和听觉上都提供了极为令人满足的反馈，这是游戏最核心的魅力。 |
| **简约美术风格** | 简洁、明快、色彩丰富，深受玩家好评。 |
| **原创音乐与音效** | 由团队成员创作，专为增强解谜成功后的满足感而设计。 |

### C. 玩家社区反馈（来自itch.io评论）

- **好评点**: 概念富有创意，手感"Juicy"，简约美术风格令人喜爱，放松治愈。
- **主要建议**: 当前关卡数量偏少且难度偏低，玩家强烈希望有更多、更难的关卡。
- **itch.io评分**: 4.8 / 5（59条评分）
