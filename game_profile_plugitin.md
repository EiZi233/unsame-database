## 2. Core Design & Gameplay (Level 1)

### A. Mechanics

Two inputs only:

| Input | Action |
| :--- | :--- |
| **Click** | Rotate plug |
| **Drag** | Move plug |

**Core loop:** Strategically plan the insertion order — larger plugs block the space below, so sequence matters.

### B. Features

| Feature | Description |
| :--- | :--- |
| **Juicy feedback** | Visually and audibly satisfying on every successful plug-in — the game's core appeal. |
| **Clean art style** | Simple, bright, colourful — consistently praised by players. |
| **Original music & SFX** | Composed by the team; designed to amplify the satisfaction of solving each puzzle. |
